Angular RxJS Examples

In Angular it is common to not use Promises for asynchronous tasks. Instead we use Rxjs with its Oberservables and Subscriptions.
Oberservables can be portrayed as streams emitting elements or errors until completion. An Observable is not just executed at its declaration. Instead it is only executed when it is subscribed to.
Note that we always should unsubscribe to all subscriptions otherwise memory leaks can occur.
